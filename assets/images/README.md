Drop your landscape images here and name them `showcase-1.jpg`, `showcase-2.jpg`, etc.

The HTML `index.html` currently references inline Unsplash images; to use local images, edit the `<figure class="frame"><img src="..."/></figure>` elements in `index.html` or replace the src URLs with `/assets/images/showcase-1.jpg`.

Tips:
- Use compressed JPEG or WebP for best performance.
- Add width and height attributes in the `<img>` tags for layout stability.
- Use `loading="lazy"` for images outside the first viewport.
