# Friendly Lawn & Garden Services

A professional booking website for exceptional lawn and gardening services.

## Features

- **Service Showcase**: Display of lawn mowing, garden maintenance, tree & shrub care, and seasonal planting services
- **Online Booking**: Complete booking form with client information collection
- **Form Validation**: Client-side validation for all form fields
- **Responsive Design**: Mobile-friendly interface
- **Contact Information**: Easy access to business contact details

## How to Use

1. Open `index.html` in a web browser
2. Browse the available services
3. Fill out the booking form with your details:
   - Full Name
   - Email Address
   - Phone Number
   - Service Address
   - Service Type
   - Preferred Date
   - Preferred Time (optional)
   - Additional Notes (optional)
4. Submit the form to receive a confirmation message

## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Local Development

Simply open `index.html` in any modern web browser. No build process or dependencies required.

Alternatively, serve with a local HTTP server:
```bash
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000`
